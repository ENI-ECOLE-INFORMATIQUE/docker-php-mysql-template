# docker-php-mysql-template
# Template Docker PHP + Apache + MySQL

Template pédagogique pour les formations DWWM / CDA de l'ENI Ecole Informatique.

## Stack

- PHP 8.3
- Apache
- MySQL 8
- phpMyAdmin
- Docker Compose

# Prérequis

Installer :

- Docker Desktop

Vérifier :

```bash
docker --version
docker compose version
